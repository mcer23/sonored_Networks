  • DEFINICIONES

    • Sonored Networks: Marca comercial propiedad de Eleny Manzano Gómez

    Acceso a Internet o conexión a Internet: es el sistema de enlace con que el computador, dispositivo móvil o red de computadoras cuenta para conectarse a Internet, lo que les permite visualizar las páginas web desde un navegador y acceder a otros servicios que ofrece Internet, como correo, mensajería instantánea, protocolo de transferencia de archivos (FTP), etcétera.
